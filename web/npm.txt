1、安装node.js
https://nodejs.org

完成后
在命令提示符输入node -v，将显示node.js版本号
在命令提示符输入npm -v，将显示npm版本号

2、安装webpack
npm install -g webpack

3、安装vue-cli
npm install -g vue-cli

4、创建项目，安装依赖
vue init webpack-simple <项目名>
cd <项目名>
npm install
npm install vue-router --save
npm install axios --save

5、运行
npm run dev

6、构建
npm run build
