修饰符

    访问
  • public 可见：任何类 方法继承:pubilc
  • protected 可见：子类或同一个包 方法继承:protected或pubilc
  • 默认 可见：同一个包 方法继承:默认或private
  • private 可见：同一个类 方法继承:不能继承
    非访问
  • static 类,方法,类变量
  • final 类,方法,变量,参数

    final修饰的类不能够被继承，修饰的方法不能被继承类重新定义，修饰的变量/参数为常量，不可修改

  • abstract 抽象类,抽象方法
  • synchronized 线程锁块
  • volatile 线程可变变量声明,读写强制使用共享内存
  • transient 不可序列化的类变量声明
