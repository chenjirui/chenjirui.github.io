List 查找元素效率高，插入删除元素效率低，会引起其他元素位置改变

  • ArrayList 索引序列 随机读取效率高 非同步
  • LinkedList 链表 插入和删除效率高
  • Vector 同步的ArrayList 多线程专用
  • Stack 后进先出的对象堆栈

检索元素效率低，删除插入效率高，插入和删除不会引起元素位置改变

  • HashSet 哈希散列集 插入删除速度快 无法控制元素出现的次序
  • LinkedHashSet 链接散列集
  • SortedSet 提供关于元素的总体排序
  • TreeSet 排序

Queue

  • ArrayDeque 大小可变数组队列
  • PriorityQueue 优先级队列 允许高效删除最小元素

Map

  • HashMap 映射表 无序 快 非同步
  • LinkedHashMap 链接散列映射表
  • SortedMap 提供关于键的总体排序
  • TreeMap 排序
  • Hashtable 同步
  • IdentityHashMap 标识散列映射表
  • WeakHashMap 弱散列映射表 自动删除无外引用键
