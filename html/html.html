表单 form

  • action 提交表单时处理数据的脚本的URL
  • method get/post

  • fieldset 组织表单元素 分组
    • legend 分组标题
    • label 控件标签
      • for=id-label
    • input 输入
      • id=id-label
      • name=data_name 服务器端字段名
      • value=default 预设的值
      • placeholder=xxx-xxx-xxxx 提示文字
      • pattern 输入格式要求
      • maxlength 允许输入的最大字符数
      • size 框的大小
      • required 要求必填
      • autofocus 加载后默认获得焦点
      • disabled 禁用
      • readonly 只读
      • type
        • text 文本框
        • password 密码框
        • email 邮件框
        • search 搜索框
        • url URL框
        • tel 电话框
        • file 文件上传
        • hidden 隐藏字段
        • radio 单选按钮
          同一组单选按钮name相同,value不同
          发送 name=value
          checked 默认选中
        • checkbox 复选框
          同一组单选按钮name相同,value不同
          checked 默认选中
        • submit 提交按钮
          如果不填写name 属性，则名-值对就不会传给脚本
          value值为显示文本
        • image 图像提交按钮
        • src=image.url
        • reset 重置按钮
    • textarea 长文本输入区域
      • id=id-label
      • name=data_name 服务器端字段名
      • cols 文本区域宽度
      • rows 文本区域高度
    • select 选择框
      • id=id-label
      • name=data_name 服务器端字段名
      • size 选择框高度
      • multiple 允许多选

      • optgroup 选项分组
        • label 子菜单标题

        • option 选项
          注意 元素内容为选项文字
          • value=option_data
          • selected 默认选中
